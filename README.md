# Vendobot

WhatsApp sales bot built with Python, Flask, SQLite and a deterministic state machine.
Uses AI only for structured JSON extraction when necessary.

## Features

- WhatsApp Cloud API webhook
- Conversation state machine
- SQLite persistence
- JSON order format
- Optional AI extractor
- Low resource usage

## Run

python app/main.py

## Tests

pytest
